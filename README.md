Fancensus revolutionises data in the video game industry, empowering informed decisions and increasing revenue across all channels.

## Requirements

Make sure you have Node and NPM installed:

```bash
node -v
# v22.11.0
```

```bash
npm -v
# 10.9.0
```

## Installation

Run the commands:

```bash
npm install
```

```bash
npm run dev
```
